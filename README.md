# RPG Combat Game (Python)

A simple text-based RPG combat game built in Python featuring turn-based battles and player choices.

## Features

* Turn-based combat system
* Player vs enemy battles
* Basic attack mechanics
* Randomized enemy stats
* Simple command-based gameplay

## How to Play

1. Run the Python file:

   ```
   python rpg_game.py
   ```
2. Follow on-screen prompts
3. Choose actions like attack during your turn
4. Defeat the enemy before your health reaches zero

## Requirements

* Python 3.x

## Notes

* This is a beginner-friendly project focused on core game logic
* Runs entirely in the terminal (CLI)
